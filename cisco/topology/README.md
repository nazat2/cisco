# topology — Desain Lab & File Packet Tracer

Folder untuk menyimpan file topologi lab kamu.

## Isi folder
- `topology-guide.md` — cara mendesain topologi lab yang baik untuk latihan tiap modul.
- `sample-topology.svg` — diagram contoh topologi lab lengkap (2 switch + 1 router + PC).
- **Taruh file `.pkt` (Cisco Packet Tracer) kamu sendiri di sini**, misalnya:
  `lab-01-basic.pkt`, `lab-02-vlan.pkt`, dst — supaya semua materi & file lab
  dalam 1 folder yang rapi.

> File `.pkt` tidak disertakan di sini karena harus dibuat dari aplikasi
> Cisco Packet Tracer (gratis, daftar di netacad.com), tapi struktur & nama
> file di atas sudah disiapkan supaya kamu tinggal isi.
