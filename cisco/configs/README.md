# configs — Kumpulan Contoh Konfigurasi Siap Pakai

Folder ini berisi contoh konfigurasi perangkat yang lebih umum/siap pakai
(gabungan beberapa topik), cocok untuk dicontek langsung waktu praktik di
Packet Tracer / GNS3 tanpa perlu buka semua modul satu-satu.

## Isi folder
- `switch-basic.cfg` — template dasar switch access layer (VLAN + trunk + management).
- `router-basic.cfg` — template dasar router (interface, static route, DHCP).
- `full-lab-topology.cfg` — gabungan konfigurasi 2 switch + 1 router untuk 1 lab utuh.

> File di sini sengaja lebih "siap pakai" dibanding contoh per-modul, supaya
> bisa langsung disesuaikan (tinggal ganti IP/VLAN) untuk lab kamu sendiri.
