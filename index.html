readmeCara pakai website ini

  1. Buka folder di sidebar kiri (klik nama folder untuk buka/tutup), lalu pilih modul di dalamnya secara berurutan (bisa dibuka lewat ikon ☰ di HP).
  2. Baca tab Teori, coba sendiri perintah di tab Config CLI pada Packet Tracer / GNS3 kamu.
  3. Centang tab Checklist Lab tiap selesai satu tugas praktik — progres otomatis tersimpan di browser (localStorage), tidak perlu login.
  4. Lupa sintaks di tengah praktik? Buka Cheat Sheet yang merangkum semua modul dalam 1 halaman.
CiscoHub — dibuat untuk belajar mandiri jaringan Cisco (setara materi awal CCNA). Seluruh materi berupa file .md / .cfg polos di folder /cisco, jadi bisa dibaca langsung di GitHub tanpa harus buka website ini.
